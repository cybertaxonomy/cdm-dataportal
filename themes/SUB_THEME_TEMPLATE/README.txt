How to use this template
------------------------

Use this template to create a subtheme from the Garland-Cichorieae theme.
For more info about creating a subtheme, see: http://drupal.org/node/225125.

It is advised that you use Drupal standards when customizing your theme.
For the Drupal CSS coding standard, see: http://drupal.org/node/302199.
For the Drupal PHP coding standard, see: http://drupal.org/coding-standards.
