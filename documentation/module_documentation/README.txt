Documentation for the Drupal 7 CDM_Dataportal modules.
Point a webbrowser to html/index.php to use the documentation.

How to generate this documentation
----------------------------------

1. Download and install doxygen,
2. Put the CDM_Dataportal module to process in your_drupal/sites/all/modules
3. Create a folder module_documentation in your_drupal/sites/all
4. Run from the command-line: doxygen doxygen-config.txt.

See for more info http://www.stack.nl/~dimitri/doxygen/
