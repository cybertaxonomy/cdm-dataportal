Converts a media uri.

Converts according to all matching conversion rules which are currently defined.

Format of a returned array element\-:

\$return\-\_\-array\mbox{[}\{type\}\mbox{]} \{ // values for \{type\}\-:(preview, webapp, media, original) \mbox{[}'size\-\_\-x'\mbox{]}, // dimensions of preview image in px \mbox{[}'size\-\_\-y'\mbox{]}, // dimensions of preview image in px \mbox{[}'uri'\mbox{]} // the uri possibly prefixed with path \mbox{[}'embed\-\_\-html'\mbox{]} // the uri embedded into some string \}


\begin{DoxyParams}[1]{Parameters}
string & {\em \$media\-Uri} & The media uri to convert.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
array Returns an associative array of the original media uri and all conversions whereas the rule type becomes the array key of the array entries. All uris are again packed into associative arrays by means to submit additional information as defined in the rules applied. 
\end{DoxyReturn}
