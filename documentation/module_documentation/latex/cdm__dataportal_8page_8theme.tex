Returns H\-T\-M\-L for a taxon page.

A wrapper function that groups available information to show by default, when a taxon page is requested by the browser. Individual themeing has to decide what this page should include (see methods beneath) and what information should go into tabs or should not be shown at all.

It is headed by the name of the accepted taxon without author and reference.
