\begin{DoxyParams}[1]{Parameters}
array & {\em \$variables} & An associative array containing\-:
\begin{DoxyItemize}
\item media\-List\-: An array of Media entities.
\item max\-Extend
\item cols
\item max\-Rows

\end{DoxyItemize}
\item alternative\-Media\-Uri\-: An array of alternative U\-R\-Is to link the images which will overwrite the U\-R\-Is of the media parts. The order of U\-R\-I in this array must correspond with the order of images in \$media\-List.
