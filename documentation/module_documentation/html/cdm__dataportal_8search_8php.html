cdm_dataportal_search_form_path_for_ws (   $ws_endpoint)

Returns a Drupal path to a search form for a CDM webservice.

For a given CDM webservice end-point, the drupal page path to the according search form is returned. cdm webservice end points are defined in constant variables like: CDM_WS_PORTAL_TAXON_FIND and CDM_WS_PORTAL_TAXON_FINDBY_DESCRIPTIONELEMENT_FULLTEXT

Parameters
string$ws_endpointThe cdm webservice endpoint for which to find the search form path.
Returns
string The Drupal path found.
cdm_dataportal_search_form_prepare (   $action_path,

Prepares a form array for a general purpose search form.

The form is used for general purpose search functionality in the dataportal. The form returned is populated with all nessecary fields for internal processing and has the textfield element $form['query'] which holds the query term. he sub tree array can be extended to contain additional search parameters.

Parameters
string$action_pathThe Drupal path to be put into the action url to which the form will be submitted.
string$search_webserviceThe cdm-remote webservice to be used, valid values are defined by the constants: FIXME.
string$query_field_default_value
string$query_field_description
string$processThe value for #process, if NULL (default), 'cdm_dataportal_search_process' is used.
Returns
array The prepared form array.
