theme_cdm_media_gallerie (   $variables)

Return HTML for a media gallery

Parameters
array$variablesAn associative array containing:
  • mediaList: An array of Media entities.
  • maxExtend
  • cols
  • maxRows
  • captionElements: An array, possible values are like in the following example: array('title', 'description', 'file', 'filename'), //to add a link to the caption: array('titlecache', '#uri'=>t('open Image'));
  • mediaLinkType: Valid values:
    • "NONE": do not link the images,
    • "LIGHTBOX": open the link in a light box,
    • "NORMAL": link to the image page or to the $alternativeMediaUri if it is defined.
  • alternativeMediaUri: An array of alternative URIs to link the images which will overwrite the URIs of the media parts. The order of URI in this array must correspond with the order of images in $mediaList.
  • galleryLinkUri: An URI to link the the hint on more images to; if NULL no link is created.
