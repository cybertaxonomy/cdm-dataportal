theme_get_nameRenderTemplate (   $variables)

Returns HTML for a name in your theme, when implemented.

This function returns currently nothing, but can be used to override how names are rendered in your theme. See the code that is commented out for an example usage. it is used in the function get_nameRenderTemplate. See that function to find out how the create template is used.
